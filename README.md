# Rutas de Lectura Pro

Versión ampliada de la aplicación.

## Contenido

- 80 libros iniciales.
- Buscador por título, autor y tema.
- Filtros por género y tema.
- Rutas calculadas por coincidencias temáticas.
- Objetivos: ampliar, contextualizar, cuestionar y aplicar.
- Favoritos, historial y valoraciones locales.
- Exportación de actividad.
- PWA instalable y funcionamiento sin conexión.

## Actualizar la versión publicada

En tu repositorio `rutas-de-lectura`:

1. Borra o reemplaza los archivos anteriores.
2. Sube todos los archivos de esta carpeta.
3. Confirma los cambios con `Commit changes`.
4. Espera uno o dos minutos.
5. Abre nuevamente GitHub Pages.

Es importante subir `books.json`; sin ese archivo no cargará el catálogo.

## Limitaciones reales

Esta versión no tiene cuentas de usuario, base de datos compartida ni inteligencia artificial externa. Las recomendaciones se calculan localmente mediante temas compartidos. Para agregar usuarios, sincronización entre dispositivos o IA hace falta una segunda etapa con backend.
